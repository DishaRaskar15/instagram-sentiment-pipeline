# instagram-sentiment-pipeline
This project helped me understand how end-to-end data analytics systems are built and deployed.

Key learnings include:

- Data Ingestion:
  Learned how to fetch real-time user reviews using APIs instead of relying on static datasets.

- Data Processing & NLP:
  Applied sentiment analysis using TextBlob to convert unstructured text into structured insights.

- Feature Engineering:
  Created additional fields such as sentiment score, rating category, and review length to enhance analysis.

- Data Storage:
  Used SQLite as a lightweight database to simulate real-world data persistence.

- Data Visualization:
  Designed an interactive Power BI dashboard to present insights in a clear and business-friendly manner.

- Deployment Pipeline:
  Understood how to manage report lifecycle using Power BI deployment pipelines (Development → Test → Production).

- End-to-End Thinking:
  Learned how different components of a data pipeline work together to transform raw data into actionable insights.
